Addition of 2-Digit Numbers with 2-Digit Numbers (Without Carrying Over)

When adding two 2-digit numbers without carrying over, you simply add the digits in each place value (ones, tens) separately.

Steps:

  1. Write the numbers one below the other.
    Make sure to align the digits in each place value (ones and tens).
  2. Add the ones place.
    Start by adding the digits in the ones place and write the sum below the line.
  3. Add the tens place.
    Next, add the digits in the tens place and write the sum below the line.

Let’s Try an Example: 45 + 31 = ?

Write the numbers one below the other.



Start with the digits in the ones place:
5 + 1 = 6.
Write 6 in the ones place of the answer.



Next, add the digits in the tens place:
4 + 3 = 7.
Write 7 in the tens place of the answer.



The final answer is 76.
